### Introduction
Ce rapport présente le travail réalisé lors de mon stage de développement d'une application mobile de gestion de point de vente (POS). L'application a été développée en utilisant le framework Flutter, permettant une expérience utilisateur native sur les plateformes Android et iOS.

### Contexte du Stage
Le stage s'est déroulé dans le cadre du développement d'une application mobile de gestion de point de vente. L'objectif principal était de créer une solution moderne et efficace pour la gestion des ventes, des stocks et des rapports commerciaux.

### Objectifs du Stage
1. Développer une application mobile de gestion POS
2. Implémenter un système d'authentification sécurisé
3. Créer des interfaces utilisateur intuitives
4. Mettre en place un système de gestion des stocks
5. Développer des fonctionnalités de reporting
6. Assurer la compatibilité multi-plateformes

### Méthodologie
#### Technologies Utilisées
- **Framework**: Flutter
- **Langage**: Dart
- **Architecture**: MVC (Model-View-Controller)
- **Gestion d'État**: StatefulWidget
- **Base de Données**: SQLite (pour la version finale)
- **Outils de Développement**: Android Studio, VS Code
